# XplainMed

**XplainMed** is an AI-powered web app that uses a **RAG (Retrieval-Augmented Generation)** model to interpret complex **radiology reports**. Users upload a medical report in PDF format, and the app explains it in **simple, layman-friendly language**, along with **relevant medical images** fetched from DuckDuckGo.

---

##  Features

-  Upload radiology report PDFs
-  Explain complex medical terms using RAG (LLM + retriever)
-  Fetch relevant images with DuckDuckGo
-  Easy-to-use Streamlit web interface


##  Tech Stack

- **LLM**: OpenAI / Azure OpenAI
- **RAG**
- **Image Search**: DuckDuckGo API
- **Frontend**: Streamlit
- **Backend**: Python
- **PDF Parsing**:PyPDF2
